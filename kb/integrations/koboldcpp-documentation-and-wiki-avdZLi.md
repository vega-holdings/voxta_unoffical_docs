# KoboldCpp Documentation and Wiki

> Link to the official KoboldCpp Wiki for detailed documentation and configuration.

KoboldCpp is a self-contained AI text-generation software for GGML and GGUF models that provides an API endpoint compatible with Voxta. For detailed information, configuration guides, and troubleshooting, users are encouraged to consult the [KoboldCpp Wiki](https://github.com/LostRuins/koboldcpp/wiki).
