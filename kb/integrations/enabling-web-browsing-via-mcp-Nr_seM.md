# Enabling Web Browsing via MCP

> Use Model Context Protocol (MCP) servers to enable web browsing capabilities.

## Answer

To enable web browsing capabilities, you can use the **Model Context Protocol (MCP)**. You need to find and configure an MCP server specifically designed for browser use. Comprehensive lists of available MCP servers can be found on GitHub.
