# Enabling or Disabling Speech Recognition via API

> Use the HTTP API to toggle speech recognition as WebSockets do not support it.

## Answer

To enable or disable speech recognition programmatically, you must use the HTTP API. The WebSockets API does not contain messages to perform this action.
