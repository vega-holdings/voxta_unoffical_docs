# Character Architect Tool Support

> Character Architect supports Voxta files and tracks issues via GitHub.

## Answer

[Character Architect](https://github.com/axAilotl/character-architect/) is a third-party tool that supports Voxta character files. Users can report issues or request enhancements via the GitHub repository. Recent updates have addressed compatibility issues regarding negative dates in Voxta files.
