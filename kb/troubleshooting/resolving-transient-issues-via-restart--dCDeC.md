# Resolving Transient Issues via Restart

> Restarting Voxta or the PC can fix random issues by clearing memory.

## Answer

If Voxta experiences unexplained behavior or performance issues, restarting the Voxta application or the host PC can often resolve the problem. This process helps clear redundant memory that may be affecting operations.
