# Animation triggers but character stands still

> Fix for animations not playing by updating the Timeline plugin and removing duplicates.

## Answer

This issue is often caused by outdated or conflicting plugins. Ensure you are using the latest version of the **Timeline** plugin and delete all other older versions or duplicates.
