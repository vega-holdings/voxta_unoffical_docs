# Test Button Error in Voxta v146/v147

> The Test button in v146/v147 may falsely report errors; verify connection via actual chat instead.

## Answer

In Voxta versions v146 and v147, the **Test** button (particularly for the Text Generation Web UI module) is known to fail and display error messages. This is often a false negative.

If the chat functionality works correctly in the **Playground** or **Chat** interface, you can ignore the Test button error.
