# Error in SSE from backend: Failed to generate text

> The 'Error in SSE from backend' usually signifies a temporary outage with the Voxta Cloud LLM provider.

## Answer

This error typically indicates a temporary issue with the Voxta Cloud LLM provider. It is a server-side issue outside of the user's control. The recommended solution is to wait for the service to stabilize.
