# Chat Error on Local Install (Missing Model)

> Fix for chat errors on local installs by manually downloading the model in settings.

## Answer

You need to navigate to the **Settings** menu and manually download the required model there.

*Note: Future builds are expected to handle this download automatically upon startup.*
