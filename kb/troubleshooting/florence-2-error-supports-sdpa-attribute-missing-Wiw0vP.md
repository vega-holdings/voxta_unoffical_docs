# Florence-2 Error: '_supports_sdpa' Attribute Missing (Fixed in Next Build)

> The '_supports_sdpa' attribute error when using Florence-2 is a known bug that has been fixed in the next Voxta update.

## Answer

When using Florence-2, you may encounter an error such as `'Florence2ForConditionalGeneration' object has no attribute '_supports_sdpa'`. This is a known issue in a specific build of the Florence-2 integration. The developers have identified and fixed the bug. The solution is to wait for and install the next Voxta update.
