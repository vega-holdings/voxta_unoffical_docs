# Testing Voxta via Cloud before Hardware Upgrade

> Use Voxta Cloud to evaluate the software before buying new hardware.

## Answer

If you are unsure about investing in a new GPU for Voxta, you can use **Voxta Cloud** to try the software first. This allows you to experience the features with an easier setup and without immediate hardware upgrades, as the processing is handled remotely.
