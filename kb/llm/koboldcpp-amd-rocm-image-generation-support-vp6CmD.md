# KoboldCpp AMD ROCm Image Generation Support

> Confirmation that KoboldCpp ROCm supports image generation on AMD GPUs.

## Answer

Yes, the ROCm version of KoboldCpp supports image generation. Users have confirmed functionality on AMD hardware (e.g., 7900XT).

**Configuration Note:**
In recent versions, you may need to manually select **ROCm** as the backend and configure the GPU layers explicitly.
