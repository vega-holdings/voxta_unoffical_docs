# Correcting STT Misspellings (Text Replacements)

> Use Voxta Utilities: Text Replacements to map and correct STT misspellings.

## Answer

You can correct Speech-to-Text (STT) misspellings using the **User Input Replacements** feature located in **Voxta Utilities: Text Replacements**.

This feature allows you to map incorrect words (e.g., "Katie") to the correct versions (e.g., "Katey"). The system replaces these words in the user input before sending it to the LLM for processing.
