# Chat Augmentations

> Chat augmentations like Vision and Autocontinue extend AI capabilities.

## Answer

**Chat Augmentations** are modules that extend Voxta's capabilities beyond standard text generation. Examples include:

*   **Vision:** Allows the AI to process visual inputs.
*   **Autocontinue:** Enables the AI to continue generating text automatically.
