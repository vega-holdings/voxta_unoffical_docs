# Configuring Chain of Thought Module

> Location and configuration modes for the Chain of Thought module.

## Answer

Navigate to **Modules** -> **Chain of Thought** -> **Configure** in the left menu.

- **Styles:** Enabling it for a style (e.g., Companion) activates it for all characters with that style.
- **Manual:** Allows you to select specific characters individually.

*Note: This is an experimental module.*
