# ExLlamaV2 Unsupported Value Configuration Error

> Resolve ExLlamaV2 configuration errors regarding unsupported values by checking presets or re-saving.

## Answer

Check your ExLlamaV2 text generation preset for any parameters set to `0.5` that might not be supported. If you cannot find the specific value or the settings look correct, try saving the preset again to refresh the configuration.
