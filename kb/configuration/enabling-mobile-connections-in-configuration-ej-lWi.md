# Enabling Mobile Connections in Configuration

> Enable mobile device access by setting "EnableConnectOnMobile": true in appsettings.json.

## Answer

To allow connections to Voxta from mobile devices, you need to modify the `appsettings.json` configuration file. Add or update the following setting:

```json
"EnableConnectOnMobile": true
```
