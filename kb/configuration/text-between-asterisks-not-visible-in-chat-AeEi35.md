# Text between asterisks not visible in chat

> Toggle asterisk visibility in Text Gen settings to reveal hidden actions.

## Answer

This is controlled by a configuration option. Check the **Text Gen** settings in Voxta for a setting that toggles whether text between asterisks is shown or hidden.
