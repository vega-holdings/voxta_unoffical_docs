# Enabling Long Term Memory for Characters

> Instructions for enabling long-term memory in the character edit menu.

## Answer

To enable long-term memory:
1. Go to **Character Edit**.
2. Select the **Memory** tab.
3. Enable **Long Term Memory**.
4. Ensure the status indicators (toggles) are green/active.
